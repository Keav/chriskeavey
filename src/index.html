Website Portfolio

I have built various websites over the years. Some are no longer active, so below are a few screenshots of some of the work I've done including a little background information on each.

I'm really glad that I went through the excersise of creating this section. I thought that some of these sites had been lost for ever! After a thorough search of my archives, I was pleased to find them burried in an old obscure web folder.

Where possible I shall put them online so the whole sites can be viewed in all their 90's glory.

Where the websites are still available or are currently in development, I have provided links to the sites or to my GitHub repo.

Swipe or use the arrows to navigate

Current Site

http://keav.net

A personal website. Always very clean and simple, I generally use this site to experiment a bit. I currently have an Octopress and a Wordpress blog hosted here.

Current Site

http://chriskeavey.com

I always want this site to appear crisp and clean, but with great functionality. All without lag, FOUC (Flash Of Unstyled Content) or any other bugginess that might detract from a silky smooth experience. Oh, and I want this across all devices! Whilst I will try to support older versions of browsers and those with javaScript disabled, I won't do it at the cost of using the latest technology.

Future Project

www.barrytouristrailway.co.uk

Created by Blue Stag Studios in Cardiff in 2011, this site has served the Railway well. I have been managing the content for this site since it was created and have also re-written the Twitter, Google Maps and Flickr plugins. This was necessary for Twitter when the API changed to use OAuth in early 2013. As technology never stands still I plan to update the underlying code, keeping much of the design but with the intention of modernising the feel and making it responsive.

Current Project

www.barryrailcentre.co.uk

Much like its sister site of the Barry Tourist Railway, this site has been flagged as requiring a priority update. The focus will be on bringing the design up to date whilst adding a slightly more commercial feel. The underlying code and structure will be modernised and it will be made responsive.
