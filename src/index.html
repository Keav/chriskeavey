Website Portfolio

I have built various websites over years so below are a few screenshots of some of the work I've done, including a little background information on each.

Where a site is still live, I have provided a direct link to the site. Where the original site is no longer active, I have made them available in an online archive.

As tempting and easy as it would be to update the code of these old sites, I have opted to keep them as unmodified as possible, whislt retaining all their functionality, so that they can be viewed in all their original glory.

Future Project

www.barrytouristrailway.co.uk

Created by Blue Stag Studios in Cardiff in 2011, this site has served the Railway well. I have been managing the content for this site since it was created and have also re-written the Twitter, Google Maps and Flickr plugins. This was necessary for Twitter when the API changed to use OAuth in early 2013. As technology never stands still I plan to update the underlying code, keeping much of the design but with the intention of modernising the feel and making it responsive.

Future Project

www.barryrailcentre.co.uk

Much like its sister site of the Barry Tourist Railway, this site has been flagged as requiring a priority update. The focus will be on bringing the design up to date whilst adding a slightly more commercial feel. The underlying code and structure will be modernised and it will be made responsive.
